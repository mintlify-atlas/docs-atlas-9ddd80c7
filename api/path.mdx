# Path

Declare a path parameter for a path operation. Path parameters are always required and are extracted from the URL path.

```python
from typing import Annotated
from fastapi import FastAPI, Path

app = FastAPI()

@app.get("/items/{item_id}")
async def read_items(
    item_id: Annotated[int, Path(title="The ID of the item to get")],
):
    return {"item_id": item_id}
```

## Parameters

<ParamField path="default" type="Any" default="...">
  Default value if the parameter field is not set. This doesn't affect Path parameters as the value is always required. The parameter is available only for compatibility.
</ParamField>

<ParamField path="alias" type="str | None" default="None">

</ParamField>

<ParamField path="title" type="str | None" default="None">
  Human-readable title for the parameter.
</ParamField>

<ParamField path="description" type="str | None" default="None">
  Human-readable description for the parameter.

</ParamField>

<ParamField path="ge" type="float | None" default="None">
  Greater than or equal validation. If set, the value must be greater than or equal to this. Only applicable to numbers.
</ParamField>

<ParamField path="lt" type="float | None" default="None">
  Less than validation. If set, the value must be less than this. Only applicable to numbers.
</ParamField>

<ParamField path="le" type="float | None" default="None">
  Less than or equal validation. If set, the value must be less than or equal to this. Only applicable to numbers.

## Examples

### Basic Path Parameter

```python
from typing import Annotated
from fastapi import FastAPI, Path

app = FastAPI()

@app.get("/items/{item_id}")
async def read_item(item_id: Annotated[int, Path()]):
    return {"item_id": item_id}
```

### Path Parameter with Metadata

```python
@app.get("/items/{item_id}")
async def read_item(
    item_id: Annotated[int, Path(title="Item ID", description="The ID of the item to retrieve")]
):
    return {"item_id": item_id}
```

### Numeric Validations

```python
@app.get("/items/{item_id}")
async def read_item(
    item_id: Annotated[int, Path(gt=0, le=1000)]
):
    return {"item_id": item_id}
```

<Note>
  The `item_id` must be greater than 0 and less than or equal to 1000.
</Note>

### String Validations

```python
@app.get("/users/{username}")
async def read_user(
    username: Annotated[str, Path(min_length=3, max_length=20, pattern="^[a-zA-Z0-9_]+$")]
):
    return {"username": username}
```

<Note>
  The `username` must be between 3 and 20 characters and can only contain alphanumeric characters and underscores.
</Note>

## Important Notes

<Warning>
  Path parameters are always required. You cannot make them optional or provide default values.
</Warning>

<Note>
  Path parameters are extracted from the URL path template defined in the route decorator (e.g., `/items/{item_id}`).
</Note>
